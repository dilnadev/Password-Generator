# Random Password Generator

A simple and responsive **Random Password Generator** built using **HTML, CSS, and JavaScript**.  
This project allows users to generate a secure random password and copy it to the clipboard with a single click.

---

##  Features

- Generate a random password instantly
- Password includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy password to clipboard
- Simple and clean user interface
- Beginner-friendly JavaScript project
